# Preview generated code as a staged action

Preview generated code as a staged action

## Usage

``` r
ravel_preview_code(
  code,
  label = "Run generated R code",
  provider = NULL,
  model = NULL
)
```

## Arguments

- code:

  R code text.

- label:

  Label for the staged action.

- provider:

  Optional provider name.

- model:

  Optional model name.

## Value

A `ravel_action` object.
