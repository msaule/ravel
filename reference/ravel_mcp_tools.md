# Normalize MCP tool definitions for provider requests

Normalize MCP tool definitions for provider requests

## Usage

``` r
ravel_mcp_tools(tools = list())
```

## Arguments

- tools:

  A list of MCP tool definitions created by
  [`ravel_mcp_tool()`](https://msaule.github.io/ravel/reference/ravel_mcp_tool.md).

## Value

A list suitable for provider payloads.
