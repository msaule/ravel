# Open an official provider documentation or key-management page

Open an official provider documentation or key-management page

## Usage

``` r
ravel_open_provider_page(
  provider = c("openai", "copilot", "gemini", "anthropic"),
  page = c("docs", "api_keys")
)
```

## Arguments

- provider:

  Provider name.

- page:

  Which page to open.

## Value

The opened URL, invisibly.
