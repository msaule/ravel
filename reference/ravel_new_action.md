# Create a new staged Ravel action

Create a new staged Ravel action

## Usage

``` r
ravel_new_action(type, label, payload, provider = NULL, model = NULL)
```

## Arguments

- type:

  Action type.

- label:

  Human-readable label.

- payload:

  Action payload.

- provider:

  Optional provider name.

- model:

  Optional model name.

## Value

A `ravel_action` object.
